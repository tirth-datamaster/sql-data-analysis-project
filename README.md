# 📊 SQL Data Analysis Portfolio Project

## 📌 Project Overview

This project demonstrates end-to-end data analysis using MySQL.  
The goal is to analyze business sales data and extract meaningful insights using SQL queries.

This project covers:
- Database creation
- Table relationships
- Data import from CSV files
- Data analysis using SQL
- Business insight generation

It is designed as a portfolio project for Data Analyst / Data Scientist roles.

---

## 🎯 Objective

To perform structured data analysis using SQL and answer important business questions such as:

- What is the total revenue?
- Which products generate the highest sales?
- Who are the top customers?
- What are monthly sales trends?
- What percentage does each product contribute to revenue?
- What is the cumulative revenue over time?

---

## 🛠 Tools & Technologies Used

- MySQL
- SQL (Joins, Aggregations, Subqueries, CTEs, Window Functions)
- CSV datasets
